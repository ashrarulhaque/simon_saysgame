Simon Games Rules

    The object of the game is to remember the pattern as far as you can and get the high score.
    Follow the below steps:
  1. Press any key to start the game.
  2. A color will flash.
  3. Press that color to repeat the sequence.
  4. The game will add another color to the sequence which you then repeat by pressing in the correct order.
  5. The game will continue to add colors after you successfully repeat the sequence back.
  6. When you don't complete a sequence successfully, the screen will blink in red and display your score
